# Paradise Nursery

**Paradise Nursery** is a React-based web application that allows users to explore and view a collection of plants in a clean and user-friendly interface. The project focuses on demonstrating core React concepts such as component-based architecture, state management, and one-way data flow.

---

## Project Purpose

The purpose of this project is to:
- Showcase the use of **React components** to build a modular UI  
- Demonstrate **props and state** for managing and passing data  
- Practice building a simple product listing интерфace using React  
- Apply modern frontend development best practices

---

## Features

- Display a list of available plants  
- Show plant details such as name, price, and description  
- Reusable and well-structured React components  
- One-way data binding using props  
- Simple, responsive user interface

---
## Goals

- Easy migration from Create React App or Vite
- As beginner friendly as Create React App
- Optimized performance compared to Create React App
- Customizable without ejecting
